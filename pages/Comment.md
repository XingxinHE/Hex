#HTML
- Use `<!--` and `-->`
- ``` html
  <!--
  <p>This is contained inside comments.</p>
  -->
  ```
  
#C++
- Single line comment
    - ```c++
      //Single line comment
      ```
    
- Block of comments
    - ```c++
      /*
       * This is a block of comments
       *
       */
      ```
