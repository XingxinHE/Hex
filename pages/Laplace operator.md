# 📝Definition
Tthe Laplace operator is a differential operator given by the [[divergence]] of the [[gradient]] of a scalar function on Euclidean space.

# ✒Notation
It is usually denoted by 3 versions.
- $\nabla\cdot\nabla$
- $\nabla ^{2}$
- $\Delta$
