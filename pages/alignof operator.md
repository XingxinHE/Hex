# 📝Definition
The **`alignof`** operator returns the alignment in [[byte]]s of the specified [[type]] as a value of type [[size_t]].

# 🗃Example
| Expression                 | Value |
| -------------------------- | ----- |
| **`alignof( char )`**      | 1     |
| **`alignof( short )`**     | 2     |
| **`alignof( int )`**       | 4     |
| **`alignof( long long )`** | 8     |
| **`alignof( float )`**     | 4     |
| **`alignof( double )`**    | 8     |
